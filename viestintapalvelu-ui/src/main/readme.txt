Tilapäinen rakenne viestintäpalvelun angular-sovellukselle,
kun en halunnut alkaa sörkkimään liikaa yhdellä kertaa.

assets
  - css => tyylitiedostot
  - img => kuvat
  - lib => kolmannen osapuolen kirjastot
  - i18n => kielisyys properties-tiedostot
email
  - html
  - js
    - controllers
    - directives
    - filters
    - routes
    - services
    - init.js
  - index.jsp => ryhmäsähköpostien etusivu
  - initpage.jsp => testikäyttöä varten, alustaa index.jsp:n tarvitsemat tiedot
reporting
  - html
  - js
    - controllers
