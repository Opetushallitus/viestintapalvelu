Så här läser du tabellen:

Dina ansökningsmål: Dina ansökningsönskemål i den ordning som du har angett i din ansökan.

Antagningsresultat: Antagningsresultatet kan vara godkänd, på reservplats, underkänd eller annullerad. Annullerad betyder att du har blivit antagen till ett högre ansökningsönskemål. Om du är på reservplats, ser du din reservplats i denna kolumn. Om din antagning är underkänd eller annullerad, kan du inte vara på reservplats.

Tilläggsinfo: får du på adressen Studieinfo.fi. Om du behöver hjälp med att tolka brevet eller har frågor som gäller antagningen och som inte klarnar via direkt kontakt till läroanstalten, kan du kontakta Studieinfos ansökningstjänst (info@studieinfo.fi eller tfn 029 533 1010).
