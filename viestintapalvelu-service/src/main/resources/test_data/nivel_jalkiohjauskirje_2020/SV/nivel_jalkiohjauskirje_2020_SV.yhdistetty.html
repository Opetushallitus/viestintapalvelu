Mer information om antagningsresultatet

Läroanstalterna bestämmer själv om på vilka grunder sökande antas till utbildningen. Du kan fråga om antagningsgrunderna direkt av läroanstalten.

Tilläggsinfo om utbildningarna får du på Studieinfo.fi. Om du behöver hjälp med att tolka brevet eller om antagningen orsakar frågor kan du kontakta läroanstalten. Du kan också skicka e-post eller ringa Studieinfos ansökningstjänst (info@studieinfo.fi eller tfn 029 533 1010).

Rätt att söka ändring

Du kan söka ändring till antagningsresultatet genom rättelseyrkan hos regionförvaltningsverket. Innan du ansöker om ändring bör du be om ett överklagbart beslut av den läroanstalt som fattat beslutet om antagningen. Du kan be om beslutet muntligt eller skriftligt. Du bör be om beslutet inom 30 dagar från och med det att du har fått detta meddelande för kännedom. Ändring i ett beslut söks genom rättelseyrkan inom 14 dagar efter att du fått antagningsbeslutet hos det regionförvaltningsverk inom vars område läroanstalten finns. Rättelseyrkan som gäller antagning till svenskspråkig utbildning riktas dock till Regionförvaltningsverket i Västra och Inre Finland. Regionförvaltningsverkens adresser hittar du i slutet av brevet. Kontakta Studieinfos ansökningstjänst vid Utbildningsstyrelsen om läroanstalten av någon orsak inte beviljar beslutet.
